# Sistema-TierPark
### ¿Cual es su función?
Es el de facilitar el registro de diversos Vehiculos de uso civil ya sea Autos, Camionetas o Motos, permitiendo un registro mas censillo y practico,
pensado para la facil implementación en otros estacionamientos

### Funciones 
Las funcionalidades del Sistema son la soguientes
1_ Registro y Extracción del Vehiculo.
2_ Realizar cambios de precio por veiculo al igual que implementar un aumento del mismo en horas picos, facil mente modifcables por el usuario.
3_ Busqueda por matricula del Vehiculo.
4_ Busqueda de espacios diponibles en el Estacionamiento.
5_ Muetras de todos los parking ya sea que algunos este vación o ocupados.

### Futuras Implementaciones
Por el momneto no se encunetra conectado a una base de datos y no cuanta con una interfaz grafica, estas cualidades seran implementadas a futuro,
se le sera implementado una base de datos local para su facil distribución al igual que una interfaz sencilla pero interactiva.
